# 🚀 Search App (Flutter + FastAPI)

## 📌 Overview
This is a full-stack search application built using:
- Flutter (Frontend)
- FastAPI (Backend)

## ✨ Features
- 🔍 Web Search (Google)
- 📺 YouTube Shortcut
- 🛒 Shopping Shortcut
- 📰 News Shortcut
- 🖼️ Image Search

## 🛠️ Tech Stack
- Flutter
- Dart
- FastAPI
- Python
